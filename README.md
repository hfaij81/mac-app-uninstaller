## Mac 卸载应用为什么需要专门的工具？

很多人觉得 Mac 的卸载够用了——拖进废纸篓就行。

其实不然。macOS 系统本身不自带完整的卸载机制。你把一个 App 移进垃圾桶时，只有 .app 本体被删了。开发者按照规范在各处存放的配置文件、缓存、日志、启动项、浏览器插件，全都原地不动。

这不是夸张。真实情况是，卸载一个大型软件（比如 Adobe 全家桶、Xcode、虚拟机软件）之后，可能有几百 MB 甚至几个 GB 的文件留在硬盘上，悄悄占着地方。

**一个专业的 Mac 卸载工具要做的，就是把这些散落在各个角落的文件一并找出来，让你决定是否删除。**

这是 App Cleaner & Uninstaller 最核心的价值——在你卸载一个 App 时，自动扫描并列出所有关联文件，打个清单让你过目，然后一键清理。
